# Security Policy

## Supported versions

The latest release receives security fixes.

## Reporting a vulnerability

Please report vulnerabilities privately, never in a public issue or pull
request:

- GitHub: use "Report a vulnerability" (private vulnerability reporting) on
  this repository, or
- Email: gnana097@gmail.com with subject "Readsmith security".

Include what you found, a reproduction, and the impact you believe it has.
You will get an acknowledgment within 72 hours. Once a fix ships we will
credit you in the release notes unless you prefer otherwise.

## Scope notes for self-hosters

Readsmith's threat model and hardening guidance (trust tiers for authored
content, CSP, key handling, rate limits) are documented at
[readsmith.dev/docs](https://readsmith.dev/docs) under Self-host, Security.
