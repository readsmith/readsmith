# Setup

Three steps to a running docs site.

<Steps>
<Step title="Install the CLI">Add Readsmith to your project with your package manager.</Step>
<Step title="Point it at your docs">A folder of Markdown with no config still builds a working site.</Step>
<Step title="Serve it">Run the dev server and open your browser.</Step>
</Steps>

<Warning title="Fonts">
The self-hosted Spectral and IBM Plex faces are wired when the font assets land. Until then, close system fallbacks are used.
</Warning>
