# Readsmith

Point Readsmith at a folder of Markdown or MDX and get a fast, accessible docs site with working navigation, search, and a first-class reading experience.

<Info title="Served by apps/web">
This page was rendered through the P1-P7 pipeline into the reading shell, then served by Next.js. Next is the serving shell, not the compiler.
</Info>

## Install

```bash title="install.sh"
curl -fsSL https://get.readsmith.dev | sh
```

## Pick your stack

<Tabs group="lang">
<Tab title="Python">
Install with `pip install readsmith`.
</Tab>
<Tab title="Node">
Install with `npm install readsmith`.
</Tab>
</Tabs>
